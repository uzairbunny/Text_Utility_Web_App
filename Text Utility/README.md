# Text Utility Web App

A vibrant, modern, and feature-rich web application for text manipulation and analysis. Built with vanilla HTML, CSS, and JavaScript featuring stunning visual effects, multiple themes, and smooth animations.

## üé® Enhanced Features

### Text Transformation
- **UPPERCASE** - Convert text to uppercase
- **lowercase** - Convert text to lowercase
- **Capitalize** - Capitalize first letter of each word
- **Remove Spaces** - Clean up extra whitespace
- **Remove Line Breaks** - Flatten multi-line text
- **Reverse** - Reverse text order
- **Copy** - Copy text to clipboard
- **Clear All** - Remove all text

### Text Statistics
- **Word Count** - Real-time word counting
- **Character Count** - With and without spaces
- **Reading Time** - Estimate reading time based on average reading speed
- **Live Updates** - Statistics update as you type with loading animations

### Enhanced User Experience
- **4 Beautiful Themes**: Default, Dark, Neon, and Sunset
- **Theme Switching** - Instant theme changes with persistent preferences
- **Glassmorphism Design** - Modern frosted glass effects with backdrop blur
- **Smooth Animations** - Elegant transitions and micro-interactions
- **Loading States** - Visual feedback during operations with shimmer effects
- **Toast Notifications** - Success and error feedback with animations
- **Floating Theme Switcher** - Elegant theme selection buttons

### Clipboard Operations
- **Copy to Clipboard** - Copy processed text with one click
- **Paste from Clipboard** - Paste text from system clipboard
- **Permission Handling** - Graceful handling of clipboard permissions

### Responsive Design
- **Mobile-First** - Optimized for all screen sizes
- **Touch-Friendly** - Large buttons and easy interactions on mobile
- **Adaptive Layout** - Grid layouts that adapt to screen size
- **Mobile Stats Toggle** - Floating button to show/hide statistics

### Keyboard Shortcuts
- `Ctrl/Cmd + Enter`: Clear text
- `Ctrl/Cmd + K`: Clear all text
- `Ctrl/Cmd + U`: Convert to uppercase
- `Ctrl/Cmd + L`: Convert to lowercase
- `Ctrl/Cmd + C`: Copy text (when no text is selected)

## üöÄ How to Run

### Option 1: Direct Browser Access (Recommended)
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. **No server setup required** - it's a static web application!

### Option 2: Local Development Server
```bash
# Navigate to the project directory
cd Text-Utility-Web-App

# Start a simple HTTP server (Python 3)
python -m http.server 8000

# Or with Python 2
python -m SimpleHTTPServer 8000

# Or with Node.js (if you have http-server installed)
npx http-server

# Open your browser and go to http://localhost:8000
```

### Option 3: Live Server (VS Code)
1. Install the "Live Server" extension in VS Code
2. Right-click on `index.html` and select "Open with Live Server"
